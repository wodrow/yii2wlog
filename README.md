# yii2wlog
yii2 wodrow log component

## Install

```html
composer require wodrow/yii2wlog "dev-master" -vv

yii migrate --migrationPath=@wodrow/yii2wlog/migrations
```

````php
'components' => [
    ...
    'wlog' => [
        'class' => \wodrow\yii2wlog\Wlog::class,
    ],
    ...
],
'bootstrap' => [..., 'wlog'],
````
